## JSONとは

- JavaScript Object Notation  
- 中身はテキスト
- "key": value
- 値は色々入る
  - String
  - number
  - boolean
  - null
  - Object
  - Array
- データ構造は配列[]とオブジェクト{}で定義

## JSON扱うときのポイント

### JSON.parse()

JSONデータからJavaScriptオブジェクトへ変換。  

### JSON.stringify()

JavaScriptオブジェクトからJSONデータにシリアライズ。
