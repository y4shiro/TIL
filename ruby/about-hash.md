## Hashとは
連想配列とも呼ばれる型。  
keyとvalueの2つが入っている。  
Array(配列)とHash(連想配列)の違いを下に示す。  

```Ruby
array = ["ruby", "python", "java"]
hash  = {lang1: "ruby", lang2: "python", lang3: "java"}
```
